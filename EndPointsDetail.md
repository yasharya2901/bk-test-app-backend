# API Endpoints

## Option Model
- **GET /options**: Get all options
- **GET /options/{id}**: Get an option by ID
- **POST /options**: Create a new option
- **PUT /options/{id}**: Update an existing option
- **DELETE /options/{id}**: Delete an option by ID
